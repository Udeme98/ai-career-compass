# AI Career Compass

AI Career Compass is a full-stack web app that guides users through a career assessment and generates personalized career recommendations using AI.

## Features

- Interactive assessment flow in the frontend
- AI-powered career result generation from the backend
- Clear result cards and roadmap guidance
- Responsive UI built with React and Tailwind CSS

## Tech Stack

- Frontend: React, Vite, TypeScript, Tailwind CSS
- Backend: Node.js, Express, TypeScript, Zod

## Project Structure

- frontend/: React + Vite client application
- backend/: Express server and assessment logic

## Getting Started

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd ai-career-compass
```

### 2. Install frontend dependencies

```bash
cd frontend
npm install
```

### 3. Install backend dependencies

```bash
cd ../backend
npm install
```

### 4. Run the applications

Frontend:

```bash
cd frontend
npm run dev
```

Backend:

```bash
cd backend
npm run start:dev
```

## Environment Variables

Create a `.env` file in the backend folder and add any required environment variables such as your AI service API key.
